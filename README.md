## Installation & Local Usage

### Clone Repository

```bash
git clone https://github.com/SEOSiri-Official/yalla-habibi.git
cd yalla-habibi
````

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment

Create `.env` file:

```
GEMINI_API_KEY=your_api_key_here
```

### Run Locally

```bash
uvicorn app:app --reload
```

Visit:

```
http://127.0.0.1:8000
```
