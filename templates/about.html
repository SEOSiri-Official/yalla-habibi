About Yalla Habibi

Yalla Habibi is a multilingual, culturally-aware voice assistant designed to bridge language, trust, and communication across global communities.

This project is part of the SEOSiri Trust Infrastructure initiative focused on building responsible, human-centric AI systems.

Mission

To enable trusted, ethical, and accessible AI interaction across languages, cultures, and digital environments.

Technical Architecture

  • FastAPI Backend
  • Generative AI Integration
  • Voice Interaction Layer (Web Speech API)
  • Cloud Deployment Infrastructure
  • Shared UI Layout System

Platform Principles

  • Human-first interaction design
  • Multilingual accessibility
  • Minimal cognitive friction
  • Privacy-aware architecture
